# Costco AI Agent

A Python-based AI agent for automating Costco-related tasks including deal scraping, content creation, and social media posting.

## Features

- **Deal Scraping**: Automatically scrape Costco deals and promotions
- **Content Generation**: Generate captions and create engaging content
- **Image Creation**: Create visual content for social media
- **Reel Creation**: Generate video reels for platforms like Instagram
- **Automated Posting**: Schedule and post content automatically
